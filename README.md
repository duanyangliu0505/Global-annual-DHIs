# Global Annual DHIs (MATLAB Implementation)

## Overview
This repository provides MATLAB code to calculate annual Dynamic Habitat Indices (DHIs) from vegetation products (e.g., MODIS, VIIRS).

## DHIs Components
- Cumulative DHI (energy availability)
- Minimum DHI (environmental stress)
- Variation DHI (ecosystem stability)

## Data
- MODIS Collection 6.1 vegetation products
- VIIRS V2 NPP vegetation products

## How to run
1. Download the vegetation product of MODIS and VIIRS from the EarthData: https://search.earthdata.nasa.gov/search?lat=-0.140625&long=0.0703125
2. Run:
   Global_Terra_NDVI_Annual_DHIs_500m.m (an example of MODIS Terra NDVI annual DHIs calculation)
